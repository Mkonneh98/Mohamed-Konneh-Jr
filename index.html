Hello, my name is Mohamed Konneh and this is really one of my first design classes for web design. My racial background is Liberian 100% and I am first generation to be born in America. I am excited about this class because I see myself doing this in the future full time because web design seems so fun and you can really be creative with your creations, as well as have a lot of people utilize it.
